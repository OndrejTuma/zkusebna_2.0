# Zkušebna Kobylisy
Vítejte v projektu kobyliské zkušebny.

## Použité technologie
- dependency management: **yarn**
- frontend: **next.js**
- state management: **mobx**
- versioning: **Git**

## Příkazy
```
npm run dev 
```
spustí vývojové prostředí na adrese localhost:3000
```
npm run build
```
vytvoří produkční build
